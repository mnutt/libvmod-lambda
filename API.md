# Varnish Module (VMOD) `lambda`

```vcl
// Place import statement at the top of your VCL file
```

### Constructor `lambda.backend(STRING function_name, STRING region, [STRING endpoint_url], [DURATION timeout], [PROBE probe], STRING probe_payload = "{}")`

Create a new Lambda backend

* `STRING function_name`:
Lambda function name or ARN
* `STRING region`:
AWS region (e.g., "us-east-1")
* `[STRING endpoint_url]`:
Optional custom endpoint URL (e.g., for LocalStack)
* `[DURATION timeout]`:
Lambda invocation timeout in seconds (default: 62)
* `[PROBE probe]`:
Health probe configuration. If specified, you should also provide probe_payload
* `STRING probe_payload`:
JSON payload to use for health checks

#### Method `STRING <object>.invoke(STRING payload)`

Invoke the Lambda function with a JSON payload and return the response
This is a synchronous call from VCL's perspective

* `STRING payload`:
JSON payload to send to Lambda

#### Method `STRING <object>.get_function_name()`

Get the Lambda function name

#### Method `STRING <object>.get_region()`

Get the AWS region

#### Method `BACKEND <object>.backend()`

Return a VCL backend built upon the Lambda backend specification
